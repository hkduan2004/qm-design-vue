## API

### ConfigProvider

| 参数   | 说明                     | 类型                            | 默认值 |
| ------ | ------------------------ | ------------------------------- | ------ |
| locale | 国际化设置               | zh-cn \| en                     | zh-cn  |
| size   | 组件尺寸                 | large \| default \| small \| '' | -      |
| theme  | 自定义主题，十六进制格式 | string                          | -      |
