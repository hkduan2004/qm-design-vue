## API

### CopyToClipboard

| 参数        | 说明                  | 类型                   | 默认值 |
| ----------- | --------------------- | ---------------------- | ------ |
| text        | 要复制到剪贴板的文本  | string                 | -      |
| showMessage | 是否显示 Message 提示 | boolean                | true   |
| onCopy      | 复制成功的回调        | function(text, result) | -      |
