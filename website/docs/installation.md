### npm 安装

:::tip
推荐使用 npm 的方式安装，它能更好地和 [webpack](https://webpack.js.org/) 打包工具配合使用。
:::

```bash
$ npm install @jiaozhiye/qm-design-vue --save
# 或者
$ yarn add @jiaozhiye/qm-design-vue --save
```
